ProtocolOOT — краткая памятка (Linux)
=====================================

Полная инструкция: файл «ПОДРОБНАЯ_ИНСТРУКЦИЯ_для_пользователя.docx» в папке data/.
Журнал изменений по версиям: «ЖУРНАЛ_ДОРАБОТОК.md» в data/.


Запуск
------
В терминале из папки программы:

  chmod +x ProtocolOOT
  ./ProtocolOOT

Или запустите ProtocolOOT из файлового менеджера.

При первом запуске рядом с программой создаются protocols.db, Protokol/, Mintrud/ и др.


Состав комплекта
----------------
  • ProtocolOOT — программа (один исполняемый файл)
  • data/ — шаблоны Word, образцы Excel, XSD Минтруда, FAQ.txt, инструкции
  • Data_base.xlsx, Programs_base.xlsx — образцы (если включены в сборку)
  • ИНСТРУКЦИЯ_папки_сборки.txt — эта памятка

Переносите на другой ПК всю папку целиком: ProtocolOOT + data/.


PDF с оформлением DOCX
----------------------
На целевом ПК нужен LibreOffice:

  sudo apt install libreoffice-writer fonts-dejavu-core fonts-liberation

Упрощённый PDF из предпросмотра использует системные шрифты DejaVu / Liberation.
