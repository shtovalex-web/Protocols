# Программа формирования протоколов (Linux)

Это копия приложения из основного репозитория, подготовленная для Linux (`linux_port/prepare.py`).

## Запуск

```bash
python3 main.py
```

Рекомендуется из каталога `linux_port/`: `./run.sh` (активирует `.venv`).

## PDF

- **С оформлением DOCX:** LibreOffice (`sudo apt install libreoffice-writer`).
- **Упрощённый из предпросмотра:** шрифты DejaVu/Liberation (`fonts-dejavu-core`, `fonts-liberation`).
